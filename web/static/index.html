启动器镜像

API 文档

本镜像服务提供以下 API 供开发者使用：

1. 获取所有版本状态

GET /api/status

返回所有启动器及其版本的详细信息。

2. 获取指定启动器状态

GET /api/status/{launcher}

返回指定启动器的所有版本信息。

3. 获取最新版本信息

GET /api/latest

返回所有启动器的最新版本号及相关信息。

4. 获取指定启动器最新版本

GET /api/latest/{launcher}

返回指定启动器的最新版本信息。

5. 触发手动扫描

POST /api/scan

触发服务器重新扫描上游仓库更新。
